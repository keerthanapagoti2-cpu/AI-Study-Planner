# AI-Based Personalized Study Planner

## Problem Statement
Students often struggle with planning their daily study schedule, leading to confusion, stress, and poor time management.

## Solution
This project provides an AI-based personalized study planner that generates a daily study schedule based on subjects and available time.

## Features
- Personalized study plan
- Subject-wise time allocation
- Break reminders
- Simple user interface

## Technologies Used
- Google Gemini AI
- Google Firebase
- Google Cloud Platform

## Future Scope
- Weekly planning
- Progress tracking
- Notifications
